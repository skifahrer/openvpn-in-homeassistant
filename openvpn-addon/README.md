# OpenVPN Manager Add-on

OpenVPN client for Home Assistant. Connect to any OpenVPN server for secure remote access.

## Features

- Automatic OpenVPN client installation
- Dashboard control via integration
- Status monitoring
- Multiple VPN profiles

## Installation

1. Add repository to Home Assistant
2. Install this add-on
3. Start the add-on
4. Install the companion integration

See [main README](../README.md) for detailed instructions.
